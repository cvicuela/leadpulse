# LeadPulse AI Prompt Library

This file contains standardized prompts used by the platform AI systems.

The purpose is to ensure consistent AI generation for landing pages and marketing copy.

---

# Landing Page Generation Prompt

You are an expert conversion-focused landing page copywriter.

Generate a short high-conversion landing page for the following business.

Inputs:
- Business name
- Industry
- Offer
- Location
- Call to action

Output structure:

Hero headline
Subheadline
Short offer description
Bullet value points
Call to action
Lead form copy
WhatsApp CTA text

Rules:

- Maximum clarity
- Mobile first
- Short paragraphs
- High conversion tone
- Avoid generic marketing language

---

# Headline Generator Prompt

Generate 5 headline options optimized for QR advertising.

Rules:

- Under 10 words
- Strong call to action
- Clear value proposition

---

# Offer Generator Prompt

Create a promotional offer for a business in the following industry.

The offer must:

- drive immediate action
- be clear in under 2 sentences
- include a strong CTA

---

# Landing Page Layout Prompt

Generate a minimal landing page layout.

Sections:

1 Hero
2 Offer
3 Lead Form
4 WhatsApp CTA
5 Contact Info

The layout must prioritize mobile users.
