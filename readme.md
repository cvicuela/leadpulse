# LeadPulse

LeadPulse is a SaaS platform that converts **QR scans from physical advertising into measurable leads** using AI-generated landing pages and analytics dashboards.

Examples of use:

• Magazine ads  
• Billboards  
• Real estate signage  
• Events  
• Restaurant promotions  

LeadPulse competes with QR tools such as Flowcode and Beaconstac, but adds:

• AI generated landing pages  
• lead capture forms  
• conversion tracking  
• advertiser dashboards  
• publisher white-label analytics  

---

## Core Flow

QR Scan  
→ Landing Page  
→ Lead Capture  
→ Analytics Dashboard
